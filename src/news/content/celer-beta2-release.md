Hey everyone, I'm happy to finally anounce the next public beta of the Celer toolbox! Beta 2 took quite some time to release because of some major roadblocks that happened but I promised that it would release January this year and here it is :D

<img src="/public/images/celer/beta2/dashboard.png" alt="The dashboard of the new Celer Public Beta 2" width="800">

## Major Changes

As previously mentioned, Celer Beta 2 now uses the bases of Fluent 2 meaning it now fits perfectly with the Windows 11 ecosystem and support dynamic theming!

Another important change is that Celer is now _fully_ translated to English. (Celer started as a school project and as such it was written in Portuguese).

Even though there hasn't been any big feature added, there has been a lot of refactoring to the core of the app to improve modularity and performance issues beta 1 had.

## What's Next?

That's obvious! Beta 3 already has a roadmap that includes some requested features as well as some of the things removed from beta 2 either because they had a high risk to ruin the release or because they were not deemed useful enough.

This roadmap is already available to everyone on our [GitHub Projects page](https://github.com/orgs/surfscape/projects/17).

So this wraps up the announcement, you can download the beta 2 by going to the [Celer project page](/celer) and clicking the blue download button and choosing beta 2 on the available version.
