## Frequently Asked Questions

### Collective

<details>
<summary>How does SurfScape make money</summary>

We at the moment don't earn anything, our products are free and we don't sell user data, this means our only revenue source is through [donations](/donate) which we are not receiving any at the moment.

Our only spending is on the domain and that is being paid off from my own pockets.

</details>

### Celer

<details>
<summary>
Does Celer work offline?
</summary>

Most features will work offline, however the current version does not come with cleaning signatures by default, which means if you want to use the cleaning module you will have to initially connect to the internet for Celer to retrive the latest signatures, since the signatures are stored locally you can then keep using Celer without internet and have the cleaning module working. If you want to update them you'll have to connect to the internet again to retrieve the latest one.

</details>

### customWin

<details>
<summary>
When will a new version release?
</summary>

The customWin suite is currently in hiatus, there are plans to rewrite all the tools however this will be a difficult task since this requires the change of frameworks that don't support the old libraries.

We are also planning to merge all the customWin tools into one but since the current versions work fine the merge project has been paused for the time being.

</details>

<details>
<summary>
The portable variant of customIcons does not open
</summary>

This is a known issue, to fix simply delete the file `customIcons.exe.config` and open customWin again, this will generate a new config and the app will open.

</details>
